# 의료 데이터셋 분석 보고서

이 폴더에는 중국 의료기관의 다양한 의료 데이터가 포함되어 있습니다. 각 파일은 특정 의료 시스템에서 추출된 데이터로 구성되어 있으며, 실제 환자 진료 과정에서 생성된 실시간 데이터입니다.

## 📊 데이터 파일 개요

### 1. 嘉和EMR数据.xlsx (6.8 MB)
**파일 설명**: 가화(嘉和) 병원의 전자 의료 기록(EMR) 데이터
- **데이터 수**: 2,321개 환자 기록
- **컬럼 수**: 19개
- **주요 컬럼**: 
  - 환자 기본 정보 (이름, 입원번호, 직업, 거주지)
  - 의료 기록 (주증, 현병력, 과거력, 신체검사)
  - 진단 정보 (주진단, 차진단, 진단코드)
  - 치료 기록 (입원기록, 퇴원기록)

**📋 실제 데이터 예제:**
```
환자명: 施木兰 (시무란)
입원번호: 464564
주증: 反复关节疼痛7年余，乏力伴四肢水肿1周。
     (관절 통증이 7년간 반복되고, 피로감과 사지 부종이 1주일간 지속됨)
주진단: 类风湿性关节炎 (류마티스 관절염)
진단코드: M06.900
현병력: 환자가 7년 전부터 양쪽 손목관절, 왼쪽 무릎관절, 양쪽 어깨관절, 
        양쪽 발목관절의 부종과 통증이 발생했으며, 아침 강직감이 동반됨...
```

**특징**: 상세한 의료 기록이 포함된 종합적인 EMR 데이터로, 환자의 전체적인 진료 과정을 추적할 수 있습니다.

### 2. PACS影像数据 去身份证.xlsx (9.0 MB)
**파일 설명**: PACS(의료영상저장전송시스템) 영상 데이터 (신분증 정보 제거됨)
- **데이터 수**: 40,051개 영상 기록
- **컬럼 수**: 23개
- **주요 컬럼**:
  - 환자 정보 (이름, 성별, 생년월일, 연령)
  - 영상 정보 (검사번호, 검사일자, 검사결론)
  - 진단 정보 (질병명, 진단시간)
  - 영상 기술적 정보 (검사표현, 보고일자)

**📋 실제 데이터 예제:**
```
환자명: 施木兰 (시무란)
검사번호: CT1599776
검사일자: 2021-11-01
검사결론: 1.两肺间质性病变伴感染，肺气肿；纵隔多发肿大淋巴结；
         双侧微量胸腔积液，对照2021.10.26片相仿，建议复查。
         2.心影增大，心包积液，对照前片心包积液稍增多，建议心超检查。
         (양폐 간질성 병변과 감염, 폐기종; 종격동 다발성 림프절 종대;
          양측 미량 흉강액, 2021.10.26 영상과 비교하여 유사함, 재검 권장)
질병명: 流行性感冒 (유행성 감기)
```

**특징**: CT, MRI 등 의료영상 검사 결과와 해석 데이터로, 영상의학 전문의의 판독 결과가 포함되어 있습니다.

## 🔍 데이터 활용 예제

### 1. 진단 패턴 분석
```python
# EMR 데이터에서 진단 패턴 분석
import pandas as pd

emr_df = pd.read_excel('嘉和EMR数据.xlsx')
diagnosis_counts = emr_df['主诊断'].value_counts()
print("상위 10개 진단:")
print(diagnosis_counts.head(10))
```

### 2. 약물 처방 분석
```python
# HIS 입원 데이터에서 약물 처방 분석
his_in_df = pd.read_excel('HIS住院.xlsx')
medication_counts = his_in_df['药品医嘱'].value_counts()
print("가장 많이 처방된 약물:")
print(medication_counts.head(10))
```

### 3. 검사 결과 추이 분석
```python
# LIS 데이터에서 검사 결과 추이 분석
lis_df = pd.read_excel('LIS 去除身份证.xlsx')
lis_df['year'] = pd.to_datetime(lis_df['INSPECTION_DATE'], 
                               format='%Y%m%d', errors='coerce').dt.year
yearly_tests = lis_df['year'].value_counts().sort_index()
print("연도별 검사 건수:")
print(yearly_tests)
```

### 4. 영상 검사 결과 분석
```python
# PACS 데이터에서 영상 검사 결과 분석
pacs_df = pd.read_excel('PACS影像数据 去身份证.xlsx', sheet_name='Sheet1')
disease_counts = pacs_df['疾病名称'].value_counts()
print("영상 검사로 발견된 주요 질환:")
print(disease_counts.head(10))
```

## 💡 데이터 활용 가능 분야

### 1. 의료 AI 모델 개발
- **진단 예측 모델**: 증상과 검사 결과를 바탕으로 질병 진단 예측
- **치료 효과 분석**: 약물 처방과 치료 결과의 상관관계 분석
- **영상 판독 AI**: CT, MRI 영상의 자동 판독 시스템 개발

### 2. 의료 데이터 마이닝
- **질병 패턴 분석**: 계절별, 연령별 질병 발생 패턴 분석
- **치료 프로토콜 최적화**: 효과적인 치료 방법 도출
- **약물 상호작용 연구**: 약물 조합의 효과성 및 부작용 분석

### 3. 의료 품질 관리
- **진료 과정 분석**: 진료 표준화 및 품질 개선
- **의료 서비스 개선**: 환자 만족도 향상을 위한 서비스 개선
- **의료 자원 최적화**: 병원 운영 효율성 증대

### 4. 의학 연구
- **임상 연구**: 새로운 치료법의 효과성 검증
- **역학 조사**: 질병의 발생 원인과 위험 요인 분석
- **의료 정책 수립**: 공중보건 정책 개발을 위한 근거 제공

## ⚠️ 주의사항

### 데이터 사용 시 준수사항
- 이 데이터는 **연구 목적으로만** 사용되어야 합니다
- 환자 개인정보 보호를 위해 추가적인 익명화가 필요할 수 있습니다
- 의료 데이터 사용 시 관련 법규 및 윤리 가이드라인을 준수해야 합니다

- 결측값이 있을 수 있으므로 데이터 정제 과정이 필요합니다
